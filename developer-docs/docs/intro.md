# Developer guide

This site is the **technical companion** to [BuildStory.Agents](https://github.com/yc-hackathon/buildstory-agents): a Next.js app where autonomous agents generate and optimize landing pages with persona targeting and multi-armed bandits.

## Who this is for

- Contributors extending agents, APIs, or section types
- Operators wiring Supabase, Redis, and OpenAI in new environments
- Anyone debugging generation, tracking, or bandit behavior

## Product overview

For goals, user flows, and feature-level behavior, start with the repository [README](https://github.com/yc-hackathon/buildstory-agents/blob/main/README.md). These pages focus on **how the code is organized**, **which endpoints exist**, and **how to change the system safely**.
